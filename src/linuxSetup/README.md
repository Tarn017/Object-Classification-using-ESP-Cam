1. alle Dateien in diesem Ordner herunterladen
2. in Ordner `.venv/` im PyCharm-Projekt ablegen
3. den Befehl `cd .venv/` im Terminal eingeben und ENTER drücken. Anschließend `bash setup_linux.sh` im Terminal eingeben und ENTER drücken.

**Falls das nicht funktioniert/ rote Fehlermeldungen wirft:**
bekannte Fehlermeldungen:
- "\r" hinten an Dateinamen
- Datei nicht gefunden

Dann versuchen:

1. den Inhalt der Dati "setup_linux.sh" komplett kopieren
2. in das PyCharm-Terminal einfügen
3. ENTER drücken und warten
